# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog 1.1.0](https://keepachangelog.com/en/1.1.0/),  
and this project aims to adhere to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0]

### Added

- Native macOS SwiftUI app for working with the `container` CLI through a focused desktop UI.
- Menu bar extra for quick status, fast reopen access, and launch-at-login support.
- First-run onboarding flow and settings for menu bar behavior, updates, and app links.
- Runtime health dashboard with compatibility checks, preflight guidance, and install-source visibility.
- Container, image, network, volume, and registry management views with detailed inspectors and actions.
- Activity center for queued, running, failed, and completed operations.
- Diagnostics export for redacted troubleshooting bundles and summaries.
- Sparkle update support, appcast scaffolding, and Homebrew tap distribution setup.
